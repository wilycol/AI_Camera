# Implementation Guide for AI_Camera

## 1. Installation
   - **System Requirements**: Python 3.8+, MongoDB Atlas account, CUDA (optional for GPU acceleration).
   - **Installation Steps**: 
      1. Clone the repository.
      2. Install dependencies: `pip install -r requirements_cpu.txt`. (Note: weights will download on first run).
      3. **Configure Atlas**: Create a `.env` file in the root directory with your `MONGO_URI`.
